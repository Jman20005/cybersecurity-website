Key Terms Explained

Authentication: The process of verifying a user's identity.

Authorization: Granting permission to access certain resources.

Encryption: Converting data into code to prevent unauthorized access.

Firewall: A security system that monitors and controls network traffic.

Malware: Malicious software designed to harm systems or steal data.

Phishing: Fraudulent attempts to obtain sensitive information.

Ransomware: Malware that locks files until payment is made.

Zero-Day: A vulnerability exploited before a fix is available.

MFA (Multi-Factor Authentication): A security method requiring multiple forms of verification.

Data Breach: Unauthorized access to confidential information.

VPN (Virtual Private Network): A secure connection over the internet that protects privacy.
